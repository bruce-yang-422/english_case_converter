# 英文命名與大小寫轉換工具

這是一個可直接在瀏覽器使用的前端小工具，用來處理英文大小寫轉換、命名格式轉換，以及字元統計與上限判定。

## 專案網址

- GitHub 倉庫：`https://github.com/bruce-yang-422/english_case_converter`
- GitHub Pages（啟用後）：`https://bruce-yang-422.github.io/english_case_converter/`

## 功能特色

- 轉大寫
- 轉小寫
- 首字母大寫
- 名稱格式轉換
- 轉為 `snake_case`
- 顯示字元統計
- 統計全形與半形字元
- 計算加權字元數
- 可設定自訂字元上限並顯示超限提示
- 支援跟隨系統、淺色、深色主題

## 專案結構

- `index.html`：主頁面
- `style.css`：介面樣式
- `app.js`：互動邏輯與轉換功能
- `theme-init.js`：頁面初始主題設定，避免畫面閃爍

## 使用方式

直接用瀏覽器開啟 `index.html` 即可使用。

## 備註

- 本專案為純靜態頁面，不需要建置流程。
- 所有功能都在瀏覽器本機執行。
